What I prompted:
How do I calculate total tackles from my defenseStats array?
----------------------------------------------------------------------
What response I recieved:
Loop through each row and add the value at index 1 (since that column stores tackles):

public int getTotalTackles() {
  int sum = 0;
  for (int[] row : stats) {
    sum += row[1];
  }
  return sum;
}
----------------------------------------------------------------------
What I prompted:
How does this enhanced for loop work?
for (int[] row : stats)
----------------------------------------------------------------------
What response I recieved:
This loop goes through each row in the 2D array.

row becomes one entire 1D array at a time.

Then row[1] accesses tackles.

It’s cleaner than writing:

for (int i = 0; i < stats.length; i++)
